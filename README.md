## Template name: Task Tracker Kobozev A.D.
## Functionality: 
* displaying a list of tasks on the main page; 
* using Django templates; 
* Basic integration with Bootstrap.
## Requirements: 
* Python 3.x 
* Django 
* Internet access to download Bootstrap (CDN) 
## Purpose: The template is designed to teach the basics of Django and can be expanded to a full-fledged task manager (adding, deleting, editing tasks).
